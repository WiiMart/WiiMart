# WiiMart Translations

This is where translations files are hosted

**Translations aren't finished yet**, but you are free to contribute

**Syntax**:

_namespace_._property_ = _string_

**Definitions**:

_namespace_: Decides the category of the string

_property_: Decides the internal name of the string

_string_: The translated string displayed on the shop

[Here](https://docs.google.com/spreadsheets/d/1FPgM9AgXX2bl-p92uL_3DsWIU-KBI2hAh5iKdrmMl74/) you can view every string and translation

**Examples**:

platform.generic = Catalog

title.points00 = Add Wii Points

[Here](https://wiilab.wiimart.org/wiimart/WiiMart/-/blob/main/translations/template.properties) you can view a template.

[Here](https://wiilab.wiimart.org/wiimart/WiiMart/-/blob/main/translations/old/) you can view the old translation files.

## Warning

Please do **NOT** use Google Translate or any translation tool as they are inaccurate.

Do **NOT** modify any filename or any property name.

Do **NOT** translate HTML elements such as <code>&lt;br&gt;</code> and <code>&amp;rsquo;</code>.

If you need to add a line break, use the HTML element <code>&lt;br&gt;</code>.

Do **NOT** modify placeholders such as {0}, {1} and {2}. If a string has one of those, be sure to include it.
